# win系统

## 查看系统位数
有以下几种方法。
1. cmd -- systeminfo
2. 运行 dxdiag 
3. 我的电脑 - 右键 - 属性  
若是64b系统，则会明确标明64位，否则就是32b。  
看处理器架构，x86对应32b，x64对应64b。
